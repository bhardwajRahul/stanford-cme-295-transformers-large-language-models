## Цель
Этот репозиторий собирает в одном месте основные понятия, рассматриваемые в курсе Stanford CME 295 Transformers & Large Language Models. Он включает:
- **Трансформеры**: самовнимание, архитектуру, варианты, методы оптимизации (разреженное внимание, низкоранговое внимание, Flash-внимание)
- **Большие языковые модели (LLM)**: промптинг, дообучение (SFT, LoRA), настройку с учётом предпочтений, методы оптимизации (группа экспертов, дистилляция, квантирование)
- **Приложения**: LLM в роли судьи, RAG, агентов, модели рассуждений (масштабирование на этапе обучения и на этапе инференса из DeepSeek-R1)
