## Циљ
Циљ овог репозиторијума је да на једном месту сажме све важне појмове који су обухваћени курсом Stanford CME 295 Transformers & Large Language Models. Укључује:
- **Трансформери**: самопажња, архитектура, варијанте, технике оптимизације (ретка пажња, пажња ниског ранга, брза пажња)
- **LLM-ови**: промптовање, фино подешавање (SFT, LoRA), подешавање на основу склоности, технике оптимизације (мешавина експерата, дестилација, квантизација)
- **Примене**: LLM као судија, RAG, агенти, модели резоновања (скалирање током тренирања и током тестирања из DeepSeek-R1)
